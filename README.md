# 건강밥상 🍱  
AI 기반 식단 추천 앱

## 소개
만성질환(고혈압, 당뇨 등) 환자를 위한 식단 관리 앱입니다.  
사용자 정보를 기반으로 AI가 맞춤형 식단을 추천하고, 음식 성분 검색 및 섭취 기록 기능을 제공합니다.

## 주요 기능
- AI 식단 추천 (Google Gemini API)
- 음식 성분 검색 (공공데이터 API 활용)
- 섭취 기록 저장 (Room DB)
- 사용자 정보 저장 (SharedPreferences)
- 게시판 기능 (Firebase Firestore)
- 이메일/구글 로그인 (Firebase Auth)
- 알림 기능 (NotificationCompat)

## UI 구성
- DrawerLayout + BottomNavigationView
- Material Design 적용
- 주요 색상: #4CAF50 (건강 이미지 강조)

## 기술 스택
- Kotlin, Android SDK
- Firebase (Auth, Firestore)
- Room DB, SharedPreferences
- 공공데이터 API
- Google Gemini API
